# 自游 — 你的私人游戏顾问

## 产品定位

一站式智能游戏推荐平台。通过了解玩家的游戏偏好、设备条件和可用时间，为其量身推荐最合适的游戏，并提供从"发现"到"购买"的完整链路。

## 目标用户

| 用户类型 | 典型场景 |
|---------|---------|
| 游戏新手 | 刚入坑，不知道从哪款游戏开始 |
| 回归玩家 | 很久没玩，想知道近几年有什么好游戏 |
| 品类探索者 | 只玩过某一类游戏，想尝试新品类但怕踩雷 |
| 设备受限玩家 | 只有特定设备（如手机/Switch），想找到适配的好游戏 |
| 时间有限的玩家 | 每天只有碎片时间，需要"短平快"的游戏推荐 |

## 核心功能

### 1. 玩家画像采集（输入）

通过一份轻量问卷构建玩家画像，信息分为**必填项**和**选填项**：

**必填项：**

- **游戏经验等级** — 纯新手 / 轻度玩家 / 中度玩家 / 硬核玩家
- **可用游戏时间** — 每天或每周可投入的游戏时长
- **游戏目的** — 竞技对抗 / 休闲放松 / 剧情沉浸 / 社交互动 / 创意建造
- **类型偏好** — 动作 / RPG / 射击 / 策略 / 模拟 / 冒险 / 体育 / 解谜 等（可多选）
- **可用设备** — 手机 / 平板 / PC / 掌机 / 主机（支持选择具体型号，如 iPhone 16、Switch、PS5）

**选填项：**

- **平台偏好** — Steam / Epic / PlayStation Store / Nintendo eShop / App Store / 其他
- **游戏新旧偏好** — 只要经典 / 只要新作 / 都可以
- **喜欢过的游戏** — 列举 1~5 款曾经喜欢的游戏，作为口味参考锚点

### 2. 智能游戏推荐（输出）

基于玩家画像生成个性化推荐列表，每款推荐游戏包含：

| 信息项 | 说明 |
|-------|------|
| **游戏名称** | 中文名 + 英文/日文原名 |
| **游戏类型** | 所属品类标签（如：开放世界 RPG、Roguelike 动作） |
| **一句话推荐理由** | 针对该用户画像的个性化推荐语 |
| **剧情/玩法简介** | 无剧透的核心玩法与世界观介绍（100 字以内） |
| **权威评分** | Metacritic / IGN / TapTap 等主流平台评分 |
| **实况视频** | 精选游戏主播的实机游玩视频链接，直观感受游戏画面与手感 |
| **购买指引** | 对应平台的购买/下载链接（如 Steam 页面、eShop 链接、淘宝实体卡带） |

### 3. 推荐结果交互

- **换一批** — 不满意可刷新推荐
- **收藏** — 感兴趣的游戏加入心愿单
- **详情展开** — 查看更多截图、用户评价、DLC 信息
- **对比** — 支持多款游戏横向对比

## 产品亮点

1. **低门槛** — 不需要懂游戏术语，问卷用大白话引导
2. **精准匹配** — 不是"热门排行榜"，而是基于个人条件的定制推荐
3. **闭环体验** — 从发现到了解到购买，一站搞定
4. **直观决策** — 视频先行，评分辅助，降低决策成本
